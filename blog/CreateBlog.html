# VuePress + GitHub

# 构建基本项目结构

  1. 安装NODE.js

https://nodejs.org/en/

  1. 创建文件夹
mkdir 文件夹名
1
  1. 进入目录
CD 文件夹名
1
  1. 初始化
npm init -y
1
  1. 将 VuePress 作为一个本地依赖安装
npm install -D vuepress
1
  1. 在项目根目录下，新建 docs 文件夹
mkdir docs
1
  1. 新建一个 markdown 文件
echo '# Hello VuePress!' > docs/README.md
1
  1. 启动项目，可在浏览器中访问 http://localhost:8080/ 即可预览效果
npm run docs:dev
1

# 基本配置

  1. 新建 .vuepress 文件夹
mkdir docs\.vuepress
1

  1. .vuepress 文件夹下新建 config.js 文件,这里的 config.js 便是一个 Vuepress 网站必要的配置文件

  2. config.js 添加如下代码

module.exports = {
  title: 'blog-demo',
  description: 'Vuepress blog demo'
}

# 首页

修改根目录 README.md 文件

---
home: true
heroImage: /vue-logo.png
heroText: blog-demo
tagline: 博客示例
actionText: 快速上手 →
actionLink: /
features:
- title: 简洁至上
  details: 以 Markdown 为中心的项目结构，以最少的配置帮助你专注于写作。
- title: Vue驱动
  details: 享受 Vue + webpack 的开发体验，在 Markdown 中使用 Vue 组件，同时可以使用 Vue 来开发自定义主题。
- title: 高性能
  details: VuePress 为每个页面预渲染生成静态的 HTML，同时在页面被加载的时候，将作为 SPA 运行。
---

# 导航栏

在 .vupress/config.js 文件添加一些导航栏链接

module.exports = {
	themeConfig: {
	  	// 你的GitHub仓库，请正确填写
	    repo: 'https://github.com/xxxxxxx/blog-demo',
	    // 自定义仓库链接文字。
	    repoLabel: 'My GitHub',
	  	nav: [
	  		{ text: 'Home', link: '/' },
	  		{ text: 'FirstBlog', link: '/blog/FirstBlog.md' }
	  	]
	}
}

# 侧边栏

配置侧边栏，在 .vupress/config.js 文件添加代码

module.exports = {
  themeConfig: {
    sidebar: [
      ['/', '首页'],
      ['/blog/FirstBlog.md', '我的第一篇博客']
    ]
  }
}

# 部署

  1. 编译：生成静态文件
npm run docs:build
1
  1. 进入生成的文件夹
cd docs/.vuepress/dist
1
  1. 初始化
git init
1
  1. 添加
git add .
1
  1. 提交
git commit -m '日志'
1
  1. 发布
git push 
或
git push -f 地址
1
2
3
  • 也可先创建一个文件夹，将内容克隆到本地，然后将docs/.vuepress/dist下内容复制到克隆文件夹，然后从克隆文件夹进行发布

# 修改或添加内容后发布

  • 编译 --> 添加 --> 提交 --> 发布
# 在创建的目录进行编译
npm run docs:build
# 转到需要发布的文件夹进行发布
git add .
git commit -m '日志'
git push
